### **How Auto-Upgrade Works**

**1. Conversation Detection**:
- Monitors `C:\Program Files\VoiceStudio\conversations\` for new files
- Detects ChatGPT conversation files (`.txt`)
- Automatically processes new conversations

**2. Upgrade Extraction**:
- Parses conversations for code blocks
- Extracts file modifications and improvements
- Detects package installation commands
- Identifies bug fixes and enhancements

**3. Safe Application**:
- Creates timestamped backups before upgrades
- Validates upgrade instructions
- Applies upgrades safely
- Restarts services if needed

**4. Monitoring & Logging**:
- Logs all upgrade activities
- Tracks success/failure rates
- Provides rollback capability
- Monitors service health

## 🆘 **Troubleshooting**

### **Installation Issues**
- Ensure you're running as Administrator
- Check Windows PowerShell execution policy
- Verify sufficient disk space and permissions

### **Service Issues**
- Check Windows Services (`services.msc`)
- Verify ports 5080-5082 are available
- Check Windows Event Viewer for errors

### **Auto-Upgrade Issues**
- Verify ChatGPT conversation files are in `C:\Program Files\VoiceStudio\conversations\`
- Check auto-upgrade logs in `C:\Program Files\VoiceStudio\logs\auto_upgrade_*.log`
- Ensure Python venv is available for upgrade scripts
- Verify file permissions for conversation directory

### **CycloneDX + VRAM Telemetry Issues**
- Verify VoiceStudio installation
- Check PowerShell execution policy
- Ensure all services are running
- Use `-LightSBOM` for faster runs without file hashing
- Verify NVIDIA GPU and drivers for VRAM telemetry
- Check Python venv availability for telemetry

### **Performance Issues**
- Ensure GPU drivers are up to date
- Check system resources (RAM, CPU)
- Verify CUDA installation for GPU acceleration
- Use `-SkipBuild` for quick handoff generation
