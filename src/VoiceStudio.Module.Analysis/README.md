# VoiceStudio.Module.Analysis

Quality metrics, diagnostics, and training module.

## Overview

This module contains all analysis-related UI panels:
- Quality dashboards
- Audio analysis
- Visualization (spectrogram, waveform)
- Training management
- Transcription
- A/B testing
