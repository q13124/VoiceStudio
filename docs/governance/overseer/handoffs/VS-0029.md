#### End-to-end durability proof (requires backend running)

**Prerequisites:**

- Backend running on `http://localhost:8000`
- Valid profile/engine available for synthesis

**Steps:**

1. **Check preflight readiness:**

   ```powershell
   curl http://localhost:8000/api/health/preflight | ConvertFrom-Json | ConvertTo-Json -Depth 10
   ```

   Expected: `{"ok": true, "checks": {"jobs_root": {"ok": true, "path": "..."}, ...}}`

2. **Synthesize a minimal voice sample:**

   ```powershell
   $synthesisRequest = @{
       text = "Test synthesis for durability proof"
       profile_id = "<valid_profile_id>"
       engine = "xtts_v2"
       language = "en"
   } | ConvertTo-Json

   $response = Invoke-RestMethod -Uri "http://localhost:8000/api/voice/synthesize" -Method POST -Body $synthesisRequest -ContentType "application/json"
   $audioId = $response.audio_id
   Write-Host "Synthesized audio_id: $audioId"
   ```

3. **Verify artifact is registered:**

   ```powershell
   $registryPath = Join-Path $env:LOCALAPPDATA "VoiceStudio\cache\audio_registry.json"
   $registry = Get-Content $registryPath | ConvertFrom-Json
   if ($registry.PSObject.Properties.Name -contains $audioId) {
       $cachedPath = $registry.$audioId.path
       Write-Host "✅ Audio ID registered at: $cachedPath"
       Write-Host "Cached file exists: $(Test-Path $cachedPath)"
   }
   ```

4. **Verify artifact is playable before restart:**

   ```powershell
   $testUrl = "http://localhost:8000/api/voice/audio/$audioId"
   $status = (Invoke-WebRequest -Uri $testUrl -Method HEAD -UseBasicParsing).StatusCode
   Write-Host "Artifact HTTP status: $status (200 = OK)"
   ```

5. **Restart backend** (stop + start)

6. **Verify artifact still retrievable after restart:**

   ```powershell
   $statusAfterRestart = (Invoke-WebRequest -Uri $testUrl -Method HEAD -UseBasicParsing).StatusCode
   Write-Host "Artifact HTTP status after restart: $statusAfterRestart (200 = OK, 404 = lost)"

   $registryAfterRestart = Get-Content $registryPath | ConvertFrom-Json
   if ($registryAfterRestart.PSObject.Properties.Name -contains $audioId) {
       Write-Host "✅ Registry reloaded correctly: audio_id still present"
       $cachedPathAfterRestart = $registryAfterRestart.$audioId.path
       Write-Host "Cached file still exists: $(Test-Path $cachedPathAfterRestart)"
   }
   ```

**Expected artifacts:**

- Registry file: `%LOCALAPPDATA%\VoiceStudio\cache\audio_registry.json` (contains `audio_id -> {path, created_at_epoch}`)
- Cached audio file: `%LOCALAPPDATA%\VoiceStudio\cache\audio\{hash_prefix}\{hash}.wav`
- Preflight response includes `jobs_root` check

**Success criteria:**

- ✅ Preflight reports `jobs_root.ok == true`
- ✅ Synthesis returns `audio_id` + `audio_url`
- ✅ Registry file contains `audio_id` entry
- ✅ Cached file exists on disk
- ✅ Artifact HTTP HEAD returns `200` before restart
- ✅ Artifact HTTP HEAD returns `200` after restart (durability proven)
- ✅ Registry file reloads with `audio_id` present after restart
