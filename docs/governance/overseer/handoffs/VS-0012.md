## Key fix

- **System DLLs removed from publish output**: `src/VoiceStudio.App/VoiceStudio.App.csproj` includes an `AfterTargets="Publish"` target (`ExcludeSystemDllsFromPublish`) that deletes:
  - `CoreMessagingXP.dll`
  - `dcompi.dll`
  - `dwmcorei.dll`
  - `marshal.dll`

This prevents the published unpackaged EXE from crash/fail-fast modes observed when those DLLs were bundled in the publish directory.
