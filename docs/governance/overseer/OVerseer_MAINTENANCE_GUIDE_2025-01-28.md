## 🚨 TROUBLESHOOTING

### Common Issues

#### Issue: Violation Scan Too Slow

**Symptoms:**
- Scan takes too long
- System performance degraded

**Solutions:**
- Optimize scan scripts
- Use parallel processing
- Cache results
- Incremental scanning

#### Issue: Too Many False Positives

**Symptoms:**
- Many false positives found
- Time wasted on verification

**Solutions:**
- Improve filtering
- Better context analysis
- Whitelist legitimate uses
- Improve detection logic

#### Issue: Worker Not Completing Tasks

**Symptoms:**
- Tasks not completed on time
- Quality issues

**Solutions:**
- Escalate to user
- Reassign tasks
- Provide guidance
- Document issues

#### Issue: Compliance Dropping

**Symptoms:**
- Compliance rate decreasing
- New violations appearing

**Solutions:**
- Run comprehensive scan
- Identify root cause
- Create fix tasks
- Monitor closely
