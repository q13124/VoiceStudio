## 🚨 Emergency Procedures

### Critical Blocker Detected

**When:** Worker cannot proceed due to blocker

**Actions:**
1. **Immediate** (within 1 hour):
   - [ ] Identify blocker
   - [ ] Assess impact
   - [ ] Notify all workers if needed

2. **Short-term** (within 4 hours):
   - [ ] Investigate root cause
   - [ ] Develop solution or workaround
   - [ ] Implement fix or provide guidance

3. **Follow-up** (within 24 hours):
   - [ ] Verify blocker resolved
   - [ ] Document solution
   - [ ] Update procedures if needed

---

### Worker Not Making Progress

**When:** Worker has not completed tasks in 2+ days

**Actions:**
1. **Check Status:**
   - [ ] Review worker's progress JSON
   - [ ] Check for blockers
   - [ ] Verify tasks are clear

2. **Investigate:**
   - [ ] Contact worker (if applicable)
   - [ ] Review assigned tasks
   - [ ] Check for technical issues

3. **Resolve:**
   - [ ] Provide clarification if needed
   - [ ] Resolve blockers
   - [ ] Reassign tasks if necessary

---

### Quality Issues Detected

**When:** Code quality issues or violations found

**Actions:**
1. **Immediate:**
   - [ ] Document violations
   - [ ] Notify worker
   - [ ] Request fix

2. **Verification:**
   - [ ] Review fix
   - [ ] Verify compliance
   - [ ] Approve or request additional fixes

3. **Prevention:**
   - [ ] Update guidelines if needed
   - [ ] Provide feedback
   - [ ] Document lessons learned
