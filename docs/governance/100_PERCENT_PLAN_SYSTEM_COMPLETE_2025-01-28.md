### For Workers

#### Starting a Task
1. Read task details in `COMPLETE_100_PERCENT_PLAN_2025-01-28.md`
2. Review requirements and acceptance criteria
3. Plan implementation approach
4. Begin implementation

#### Completing a Task
1. Implement all requirements
2. Meet all acceptance criteria
3. Test thoroughly
4. Fill out `TASK_COMPLETION_REPORT_TEMPLATE_2025-01-28.md`
5. Submit completion report to Overseer
6. Wait for verification

#### Daily Work
1. Work on assigned tasks
2. Update progress in `TASK_TRACKING_TEMPLATE_2025-01-28.md`
3. Fill out `DAILY_PROGRESS_REPORT_TEMPLATE_2025-01-28.md` at end of day
4. Report blockers immediately

### For Overseer

#### Verifying Tasks
1. Receive completion report from worker
2. Review completion report
3. Use `OVERSEER_VERIFICATION_CHECKLIST_2025-01-28.md`
4. Verify all requirements and acceptance criteria
5. Scan for violations
6. Review code and tests
7. Mark as VERIFIED or NEEDS REVISION

#### Daily Monitoring
1. Review daily progress reports
2. Update `TASK_TRACKING_TEMPLATE_2025-01-28.md`
3. Track blockers
4. Generate progress summary
5. Report to stakeholders
