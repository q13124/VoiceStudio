- **Feature Name** - Name of the feature
- **v[FROM_VERSION]** - Status/value in previous version
- **v[TO_VERSION]** - Status/value in new version
- **Status** - Same/Updated/New/Improved/Deprecated/Removed

### Status Indicators

- ✅ = Available
- ❌ = Not Available
- **NEW** = New in this version
- **IMPROVED** = Enhanced in this version
- **UPDATED** = Updated with changes
- **DEPRECATED** = Deprecated (will be removed)
