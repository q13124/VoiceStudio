# Screenshots Directory

This directory is intended for application screenshots to be used in documentation.

## Screenshot Requirements

### Getting Started Guide
- Main application window
- First launch dialog
- Profile creation screen
- Voice synthesis interface

### User Manual
- All panel screenshots
- Timeline view
- Effects mixer
- Analyzer view
- Macro editor
- Training module

### Tutorials
- Step-by-step screenshots for each tutorial
- Before/after examples where applicable

## Screenshot Guidelines

1. **Resolution:** Minimum 1920x1080, recommended 2560x1440 or higher
2. **Format:** PNG format preferred
3. **Naming:** Use descriptive names (e.g., `main-window.png`, `profile-creation.png`)
4. **Annotations:** Add annotations if needed to highlight features
5. **Privacy:** Ensure no sensitive data visible in screenshots

## Current Status

**Status:** ⚠️ Screenshots not yet captured

**Note:** Screenshots should be captured from the actual running application. This directory is prepared for when screenshots are available.

## Adding Screenshots

1. Capture screenshot from running application
2. Save to appropriate subdirectory
3. Reference in documentation using relative path:
   ```markdown
   ![Main Window](screenshots/main-window.png)
   ```
