# VoiceStudio Feature Guide

Overview of core features and workflows.

---

## Voice Synthesis

- **Engines**: XTTS, Piper, Bark, and more. Select engine in the Voice Synthesis panel.
- **Profiles**: Create voice profiles from reference audio for cloning.
- **SSML**: Use SSML tags for prosody control (pitch, rate, emphasis).
- **Batch**: Synthesize multiple segments or files in one operation.

---

## Transcription

- **Engines**: Whisper, Vosk, and others. Choose in the Transcription panel.
- **Languages**: Multi-language support with auto-detection.
- **Timestamps**: Word-level or segment-level timestamps.
- **Export**: Export transcripts as SRT, VTT, or plain text.

---

## Voice Profiles

- **Create**: Record or import reference audio, then create a profile.
- **Manage**: View, edit, and organize profiles in the Profiles panel.
- **Health**: Profile health dashboard shows quality and readiness.

---

## Effects Mixer

- **Effects**: Compressor, reverb, EQ, noise reduction, and more.
- **Chain**: Apply multiple effects in sequence.
- **Presets**: Save and load effect chains.
- **Real-time**: Preview effects before applying.

---

## Timeline

- **Tracks**: Multi-track editing with waveforms.
- **Regions**: Cut, copy, paste, and move regions.
- **Automation**: Envelope automation for volume and effects.
- **Export**: Export the full mix or selected regions.

---

## Plugin Installation

1. Open **Plugin Gallery** from the menu or panel.
2. Browse or search for plugins.
3. Click **Install** on a plugin.
4. Restart VoiceStudio if prompted.
5. Installed plugins appear in the appropriate panels (Effects, Engines, etc.).

---

## Diagnostics Panel

- **Health**: Backend connection, engine status, GPU info.
- **Logs**: View application and error logs.
- **Model Drift**: Monitor model quality drift (Phase 9).
- **Traces**: Request tracing for debugging.
