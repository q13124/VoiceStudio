### Application

|----------|--------|-------------|

| `Esc` | Cancel/Close | Cancel operation or close dialog |

## Tips

1. **Context-Sensitive:** Many shortcuts are context-sensitive and change based on the active panel or selection.

2. **Tooltips:** Hover over buttons and controls to see their keyboard shortcuts in tooltips.

3. **Command Palette:** Press `Ctrl+P` to access all commands via the command palette.

4. **Help:** Press `F1` in any panel to see panel-specific help and shortcuts.

5. **Customization:** Keyboard shortcuts can be customized in Settings (future feature).
