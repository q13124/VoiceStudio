## 🎯 Panel-Specific Shortcuts

### Pronunciation Lexicon
- **Ctrl+N** - Add Entry
- **Ctrl+S** - Search
- **Ctrl+E** - Estimate Phonemes
- **Delete** - Delete Entry
- **F5** - Refresh

### Script Editor
- **Ctrl+N** - New Script
- **Ctrl+S** - Save Script
- **Delete** - Delete Segment

### Transcription
- **Ctrl+T** - Start Transcription
- **F5** - Refresh

### Voice Synthesis
- **Ctrl+Enter** - Synthesize
- **Enter** - New Line

---

## 💡 Tips

1. **Context Matters:** Some shortcuts (like **S** and **M**) change based on which panel has focus
2. **Command Palette:** Press **Ctrl+P** to quickly find any command
3. **Multi-Select:** Use **Ctrl+Click** and **Shift+Click** for batch operations
4. **Undo/Redo:** Use **Ctrl+Z** and **Ctrl+Y** frequently
