# VoiceStudio Quick Start Guide

Get up and running with VoiceStudio in minutes.

---

## 1. Installation

1. Download `VoiceStudio-Setup-v1.0.2.exe` from the release.
2. Run the installer. Accept the license and choose install location.
3. The installer will prompt for prerequisites (e.g., .NET 8 Desktop Runtime, Python) if needed.
4. Complete the setup wizard.

---

## 2. First Launch

1. Launch VoiceStudio from the Start Menu or Desktop shortcut.
2. Wait for the main window to open (typically within 30 seconds).
3. The backend starts automatically. The status bar shows **Connected** when ready.
4. On first run, you may see a welcome wizard—follow the prompts or skip.

---

## 3. Import Audio

1. **File → Import Audio** or drag a WAV/MP3 file onto the Library.
2. Supported formats: WAV, MP3, FLAC, OGG, M4A, AAC, AIFF, WMA.
3. The file appears in the Library panel.

---

## 4. Basic TTS Workflow

1. Open the **Voice Synthesis** panel.
2. Select a voice profile (or create one from a reference clip).
3. Enter text in the synthesis editor.
4. Click **Synthesize**.
5. The generated audio appears in the timeline or Library.
6. **Export** to save as WAV or MP3.

---

## 5. Basic Transcription

1. Select an audio asset in the Library.
2. Open the **Transcription** panel.
3. Choose an engine (e.g., Whisper).
4. Click **Transcribe**.
5. View and edit the transcript.
