## ✅ HOW TO REMOVE EXTENSIONS

### Method 1: Extensions View (Recommended)
1. Press `Ctrl+Shift+X` to open Extensions view
2. Search for each extension ID (e.g., `dbaeumer.vscode-eslint`)
3. Click the gear icon next to the extension
4. Select "Disable (Workspace)" or "Uninstall"
5. Repeat for all extensions in this list

### Method 2: Command Palette
1. Press `Ctrl+Shift+P` to open Command Palette
2. Type "Extensions: Show Installed Extensions"
3. Search and disable/uninstall from the list

### Method 3: Bulk Removal (PowerShell)
```powershell
# List of extension IDs to remove (example)
$extensionsToRemove = @(
    "dbaeumer.vscode-eslint",
    "esbenp.prettier-vscode",
    "bradlc.vscode-tailwindcss"
    # Add more extension IDs here
)

# Remove each extension
foreach ($ext in $extensionsToRemove) {
    code --uninstall-extension $ext
}
```
