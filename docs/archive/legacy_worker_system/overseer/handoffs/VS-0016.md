## Changes

### `src/VoiceStudio.App/Core/Engines/` (New Directory)

- **`IEngine.cs`**: Base interface for all engines.

  - Properties: `Id`, `Name`, `Version`, `Capabilities`.
  - Methods: `InitializeAsync`, `ShutdownAsync`, `IsAvailableAsync`.
  - Enum: `EngineCapabilities` (Flags).

- **`ITextToSpeechEngine.cs`**: Interface for TTS engines.

  - Inherits `IEngine`.
  - Methods: `GetVoicesAsync`, `SynthesizeAsync`.

- **`ITranscriptionEngine.cs`**: Interface for Speech-to-Text engines.
  - Inherits `IEngine`.
  - Methods: `TranscribeAsync`.
