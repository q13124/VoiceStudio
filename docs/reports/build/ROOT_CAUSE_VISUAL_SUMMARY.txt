## The Fixes (Quick Reference)

### FIX #1: Rename Class (1 line)
```
File: TextSpeechEditorViewModel.cs, Line 640
─────────────────────────────────────────────

BEFORE:
    public class TextSpeechEditorSegmentItem

AFTER:
    public class TextSegmentItem

Result: ✓ Eliminates ~80 errors
```

### FIX #2: Add Using Statement (1 line per file)
```
File: TrainingQualityVisualizationViewModel.cs (and others)
Line: Top of file
─────────────────────────────────────────────────────────

BEFORE:
    using VoiceStudio.Core.Services;
    // missing: using VoiceStudio.Core.Models;
    using VoiceStudio.App.Services;

AFTER:
    using VoiceStudio.Core.Services;
    using VoiceStudio.Core.Models;          ← ADD THIS
    using VoiceStudio.App.Services;

Result: ✓ Eliminates ~40-50 errors
```

### FIX #3: Debug XAML Bypass
```
File: VoiceStudio.App.csproj
─────────────────────────────────────────────────────

Option A - Verify Import Placement
  ✓ Check that custom targets import is before Sdk attribute

Option B - Use Property-Based Disable
  Add to <PropertyGroup>:
    <DisableXbfGeneration>true</DisableXbfGeneration>
    <EnableUIXamlCompilation>false</EnableUIXamlCompilation>

Result: ✓ Enables build to proceed
```
