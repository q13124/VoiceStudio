## Features

### Primary Visualizations

1. **Quality Heatmap**
   - X-axis: Time (sessions/days)
   - Y-axis: Voice profiles
   - Color: Quality score (red → yellow → green)
   - Click cell for details

2. **Engine Comparison**
   - Bar chart comparing engines
   - Metrics: MOS, Similarity, Naturalness, SNR
   - Grouped by metric type

3. **Trend Analysis**
   - Line chart over time
   - Selectable metrics
   - Moving average overlay

4. **Anomaly Detection**
   - Alert cards for quality drops
   - Threshold indicators
   - Suggested actions

### API Integration

Existing endpoints:
- `POST /api/quality/analyze` - Analyze audio quality
- `POST /api/quality/compare` - Compare multiple samples
- `POST /api/quality/benchmark` - Benchmark across engines
- `GET /api/quality/history/{profile_id}` - Quality history
- `GET /api/quality/trends` - Quality trends
- `GET /api/quality/consistency/{project_id}` - Project consistency

### Quality Metrics

| Metric | Range | Good Threshold |
|--------|-------|----------------|
| MOS | 1.0 - 5.0 | > 3.5 |
| Similarity | 0.0 - 1.0 | > 0.7 |
| Naturalness | 0.0 - 1.0 | > 0.6 |
| SNR | dB | > 20 dB |
| PESQ | -0.5 - 4.5 | > 3.0 |
| STOI | 0.0 - 1.0 | > 0.8 |
