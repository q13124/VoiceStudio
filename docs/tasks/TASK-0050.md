# TASK-0050: BaseEngine to EngineProtocol Migration

## Overview

Migrate legacy engines from using the `BaseEngine` alias to the canonical `EngineProtocol` interface. This completes TD-017 from the FUTURE_WORK registry.

## Background

`BaseEngine` was introduced as a backward-compatible alias for `EngineProtocol` to avoid breaking existing engine implementations during the protocol introduction. Now that the protocol is stable, all engines should reference `EngineProtocol` directly.
