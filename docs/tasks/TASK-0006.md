### API Integration

Existing endpoint: `POST /api/voice/ab-test`

```json
{
  "text": "Sample text to synthesize",
  "config_a": { "engine": "xtts", "profile_id": "..." },
  "config_b": { "engine": "piper", "profile_id": "..." }
}
```

Response includes:
- `audio_a`: Base64 audio or path
- `audio_b`: Base64 audio or path
- `metrics_a`: MOS, similarity, naturalness, SNR, artifacts
- `metrics_b`: MOS, similarity, naturalness, SNR, artifacts
- `overall_winner`: "a" | "b" | "tie"
