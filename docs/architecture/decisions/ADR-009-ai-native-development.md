# ADR-009: AI-Native Development Patterns

## Status

Proposed

## Context

VoiceStudio is developed with AI agent assistance. Patterns are needed to maximize AI effectiveness while maintaining code quality and human oversight.

## Options Considered

1. **Traditional development** - AI as code completion only
2. **AI-assisted** - AI suggests, human implements
3. **AI-native** - Role-based agents with governance
