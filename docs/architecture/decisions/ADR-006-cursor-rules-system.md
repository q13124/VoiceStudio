# ADR-006: Enhanced Cursor Rules System

## Status

Proposed

## Context

VoiceStudio uses Cursor IDE with agent-based development. A comprehensive rules system is needed to guide agent behavior, enforce conventions, and maintain quality.

## Options Considered

1. **Single AGENTS.md** - All rules in one file
2. **Modular .mdc rules** - Categorized rules in `.cursor/rules/`
3. **Hybrid approach** - AGENTS.md index with modular rules
