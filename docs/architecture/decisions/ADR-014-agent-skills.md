# ADR-014: Agent Skills Integration

## Status

Proposed

## Context

Cursor agents can use skills to enhance their capabilities. A skills system is needed to define, discover, and invoke project-specific skills.

## Options Considered

1. **No skills** - Agents use only built-in tools
2. **SKILL.md pattern** - Skills as markdown with instructions
3. **MCP tools** - Skills as MCP server tools

## Decision

**To be decided.** This ADR is a placeholder.

Implementation evidence:
- `.cursor/skills/` - Project skills
- `SKILL.md` files with role-specific capabilities
