# API JSON Schemas

This directory contains JSON Schema definitions for VoiceStudio Quantum+ API requests and responses.

## Schema Files

- `voice_profile.schema.json` - Voice profile data structure
- `profile_create_request.schema.json` - Create profile request
- `voice_synthesize_request.schema.json` - Voice synthesis request
- `voice_synthesize_response.schema.json` - Voice synthesis response
- `project.schema.json` - Project data structure

## Usage

These schemas can be used for:
- API validation
- Code generation
- Documentation
- Testing

## Schema Format

All schemas follow JSON Schema Draft 7 specification.

## Adding New Schemas

When adding new API endpoints, create corresponding schema files:
1. Create `{endpoint_name}.schema.json`
2. Follow existing schema patterns
3. Include all required fields
4. Document all properties
5. Provide examples
